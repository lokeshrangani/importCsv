## About Project

To Read Any csv file and import in database:

- [Clone Repository](https://github.com/lokeshrangani/importCsv).
- Rename '.env.example' to '.env'.
- Run Migration [php artisan migrate].
- php artisan serve.
- [Go to](127.0.0.1:8000)
